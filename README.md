# Javascript-Calculator

This is a simple JavaScript calculator web application that allows you to perform basic arithmetic operations (addition, subtraction, multiplication, and division).

## Features

- **User-Friendly Interface**: The calculator provides a user-friendly interface with input fields for two numbers and a dropdown to select the operation.

- **Four Arithmetic Operations**: You can perform addition, subtraction, multiplication, and division.

- **Error Handling**: The calculator handles division by zero and displays an error message in such cases.

## Technologies Used

- HTML
- JavaScript
- Bootstrap (for styling)

## Usage

1. Enter the first number in the "Number-1" input field.

2. Choose an operator (+, -, *, /) from the dropdown.

3. Enter the second number in the "Number-2" input field.

4. Click the "Submit" button to calculate the result.

5. The result will be displayed in the "Answer" field.

## How to Run

You can open the `index.html` file in a web browser to use the calculator. No additional setup is required.
